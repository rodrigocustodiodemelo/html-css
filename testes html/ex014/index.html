main css

configurações para a tag main em css background-color (utilizamos essa tag para alterar a cor do plano de fundo,seja de um elemento ou do documento como um todo . para isso, basta atribuir a cor desejada á propriedade backound-color ,declarada dentro do seletor no qual ela deve ser aplicada. ) width:600px;( essa tag determina a largura da área de conteúdo de elemento .a área de conteúdo fica dentro do preenchimento ,da borda , e da margem de um elemento.) pedding:10px;(essa tag define uma distância entre o conteúdo de um elemento e suas bordas. é um atalho que evita definir uma distância para cada lado separadamente ) margin:auto;(essa tag define que o navegador ira definir a margem) border-radius:10px(essa tag permite arredondar as bordas de um elemento) box-shadow:10px,10px,10px,cor(é utilizado para adicionar efeitos de sombra em volta de um elemento)
